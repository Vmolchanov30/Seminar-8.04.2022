## Создание фиксаций

Для того, чтобы зафиксировать изменения в текущей версии используется команда *git commit*. Используется она следующим образом: *git commit -m <Сообщение к коммиту>*. Сообщение к коммиту писать **ОБЯЗАТЕЛЬНО**
